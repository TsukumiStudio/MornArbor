## 概要

Arbor 向けのユーティリティ State / Process / Tween / SubState 拡張集。ポインタ・入力待機・シーン遷移・Tween制御などを手軽に扱える。

## 導入方法

Unity Package Manager で以下の Git URL を追加:

```
https://github.com/TsukumiStudio/MornArbor.git?path=src#1.0.1
```

`Window > Package Manager > + > Add package from git URL...` に貼り付けてください。
